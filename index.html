Poem Generator.

Generating random poems, from a defined Grammar.

The poem-generator project, consists of a solution of a programming exercise to create a random poem generator using the following grammatical rules:

POEM: <LINE> <LINE> <LINE> <LINE> <LINE> LINE: <NOUN>|<PREPOSITION>|<PRONOUN> $LINEBREAK ADJECTIVE: black|white|dark|light|bright|murky|muddy|clear <NOUN>|<ADJECTIVE>|$END NOUN: heart|sun|moon|thunder|fire|time|wind|sea|river|flavor|wave|willow|rain|tree| flower|field|meadow|pasture|harvest|water|father|mother|brother|sister <VERB>|<PREPOSITION>|$END PRONOUN: my|your|his|her <NOUN>|<ADJECTIVE> VERB: runs|walks|stands|climbs|crawls|flows|flies|transcends|ascends|descends| sinks <PREPOSITION>|<PRONOUN>|$END PREPOSITION: above|across|against|along|among|around|before|behind|beneath|beside|between| beyond|during|inside|onto|outside|under|underneath|upon|with|without|through <NOUN>|<PRONOUN>|<ADJECTIVE>

These rules are load from a file, and can be changed at any time, inside the project you can find this file in /main/resources/GrammarInput.txt

For getting a local copy of this project, you could try to clone the repo as follows:

  1. $ cd workspace/
  2. $ mkdir poem-generator
  3. $ git clone https://github.com/m4solutions/poem-generator.git
  4. $ git fetch origin
If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
